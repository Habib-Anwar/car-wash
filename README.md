This is a simple application built with Express, dotenv, cors, Mongoose,json web token and TypeScript.

## Features

- User can create profile by signing up.
- User can login.
- User can book a service.
- Admin can create services.
- Admin can update services.
- Admin can delete services.
- Admin can create slots.
- Admin can get all the bookings.
- Validation using Zod.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js (>= 14.x)
- npm
- MongoDB (running locally or accessible remotely)
- postMan
